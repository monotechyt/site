LeetDown

an OTA downgrade utility for A6 & A7 devices



Download Latest Release (2.2) See older releases on GitHub



What is LeetDown?
LeetDown is a GUI app powered by libimobiledevice that allows downgrading compatible A6 and A7 devices to OTA signed firmwares.
